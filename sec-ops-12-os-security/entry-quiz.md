# Entry Quiz — Session 12: OS Security (Windows and Linux)

**Instructions:** Choose the single best answer for each question.
This quiz checks prerequisite knowledge before the session.

---

**Question 1:** What does "SUID" stand for in Linux?

A) System User ID
B) Set User ID
C) Secure Unix ID
D) Superuser Interface Daemon

**Correct Answer: B**

---

**Question 2:** In Windows, what is the "registry"?

A) A list of all installed software
B) A hierarchical database that stores OS and application configuration settings
C) A file that contains all user passwords in encrypted form
D) The Windows security event log

**Correct Answer: B**

---

**Question 3:** What is the purpose of a firewall?

A) To scan files for viruses and malware
B) To encrypt network traffic between systems
C) To control which network connections are allowed or denied based on rules
D) To monitor user activity and log all actions

**Correct Answer: C**

---

**Question 4:** In Linux, which file contains user account information including usernames and home directories?

A) /etc/shadow
B) /etc/passwd
C) /etc/users
D) /etc/accounts

**Correct Answer: B**

---

**Question 5:** What is the principle of "least privilege"?

A) Using the simplest password possible
B) Only granting users the minimum access necessary to perform their work
C) Ensuring all users have equal access to all systems
D) Requiring administrators to use a separate computer

**Correct Answer: B**

---

**Question 6:** Which Windows process is responsible for managing user authentication and storing credential hashes in memory?

A) svchost.exe
B) winlogon.exe
C) lsass.exe
D) explorer.exe

**Correct Answer: C**

---

**Question 7:** In Linux, what does a file permission of `chmod 777` grant?

A) Read and write for the owner only
B) Read, write, and execute for the owner, group, and all others
C) No access for any user
D) Full access for root only

**Correct Answer: B**

---

**Question 8:** What is the purpose of the Linux `/etc/shadow` file?

A) To store shadow copies of the filesystem
B) To store encrypted/hashed passwords for user accounts
C) To define sudo permissions
D) To list all running system daemons

**Correct Answer: B**

---

**Question 9:** What is "sudo" in Linux?

A) A command that switches the user to root permanently
B) A tool that allows permitted users to run commands as another user (typically root)
C) A system daemon that manages user accounts
D) A file that stores root's password

**Correct Answer: B**

---

**Question 10:** What is a Windows Event Log used for?

A) Storing configuration settings for installed applications
B) Recording system events, errors, security events, and application activity
C) Managing scheduled maintenance tasks
D) Storing backup copies of system files

**Correct Answer: B**

---

**Question 11:** Which of the following best describes "Discretionary Access Control" (DAC)?

A) Access is controlled by the operating system kernel regardless of user settings
B) The owner of a resource decides who has access to it
C) Access is granted based on mandatory labels set by a security administrator
D) All users share the same level of access

**Correct Answer: B**

---

**Question 12:** What is the primary purpose of auditing in an OS security context?

A) To improve system performance
B) To track and record security-relevant events for detection and forensics
C) To encrypt user data
D) To manage software updates

**Correct Answer: B**
