# Entry Quiz — Session 11: Introduction to VERIS

**Instructions:** Choose the single best answer for each question.
This quiz assesses prerequisite knowledge needed to get the most from this session.

---

**Question 1:** What does the acronym CIA stand for in cybersecurity?

A) Computer Incident Analysis
B) Confidentiality, Integrity, Availability
C) Classification, Investigation, Assessment
D) Cyber Intelligence Agency

**Correct Answer: B**

---

**Question 2:** In the context of cybersecurity, what is a "data breach"?

A) Any unauthorized access attempt to a network
B) A confirmed incident where sensitive data is disclosed to an unauthorized party
C) A virus that damages files on a hard drive
D) A denial-of-service attack that makes a website unavailable

**Correct Answer: B**

---

**Question 3:** Which of the following is an example of a "social engineering" attack?

A) Exploiting a SQL injection vulnerability in a web application
B) Installing malware through a USB drive
C) Sending a fake email impersonating IT support to steal credentials
D) Brute-forcing a user's password

**Correct Answer: C**

---

**Question 4:** What is a JSON file?

A) A type of database that stores data in tables
B) A compiled binary executable program
C) A text-based data format using key-value pairs
D) A network packet capture format

**Correct Answer: C**

---

**Question 5:** Which of the following best describes an "insider threat"?

A) A hacker who penetrates a network from a foreign country
B) A malware program that spreads through email attachments
C) A risk from employees or other trusted insiders who misuse their access
D) A physical intrusion into a secure facility by an outsider

**Correct Answer: C**

---

**Question 6:** What does the term "threat actor" refer to in cybersecurity?

A) A software vulnerability that can be exploited
B) An individual, group, or entity that carries out or enables a cyber attack
C) A firewall rule that blocks malicious traffic
D) A security analyst who investigates incidents

**Correct Answer: B**

---

**Question 7:** Which of the following is an example of a "ransomware" attack?

A) An attacker intercepts network traffic to steal credentials in transit
B) An attacker floods a web server with traffic to make it unavailable
C) An attacker encrypts a victim's files and demands payment to restore access
D) An attacker creates a fake website that looks like a banking portal

**Correct Answer: C**

---

**Question 8:** Which of the following best describes the term "vulnerability" in cybersecurity?

A) A successful attack that has already compromised a system
B) A weakness in a system that could be exploited by a threat actor
C) A type of malware designed to steal sensitive data
D) A security control that prevents unauthorized access

**Correct Answer: B**

---

**Question 9:** What is the primary purpose of a Security Operations Center (SOC)?

A) To develop new security software and tools
B) To monitor, detect, and respond to cybersecurity threats and incidents
C) To conduct penetration tests of external systems
D) To manage compliance documentation and audits

**Correct Answer: B**

---

**Question 10:** What is an "incident response" process?

A) The process of installing and configuring security software
B) The process of training employees in security awareness
C) The organized approach to addressing and managing the aftermath of a security incident
D) The process of assessing third-party vendor security

**Correct Answer: C**
