# Final Quiz — Session 11: Introduction to VERIS

**Instructions:** Choose the single best answer for each question.

---

**Question 1:** What does VERIS stand for?

A) Vulnerability Event Reporting and Intelligence Sharing
B) Vocabulary for Event Recording and Incident Sharing
C) Verified Enterprise Risk Incident Schema
D) Variable Event Response and Incident System

**Correct Answer: B**

---

**Question 2:** Which of the following is NOT one of the four dimensions in the VERIS 4-A framework?

A) Actor
B) Action
C) Asset
D) Alert

**Correct Answer: D**

---

**Question 3:** A developer accidentally misconfigures a cloud database to be publicly accessible, exposing 500,000 customer records.
What is the correct VERIS action type?

A) Hacking
B) Misuse
C) Error
D) Physical

**Correct Answer: C**

---

**Question 4:** In VERIS, what does the `data_disclosure` field inside the `confidentiality` attribute section represent?

A) Whether the organization disclosed the breach to regulators
B) Whether an unauthorized party is confirmed to have accessed the data
C) The type of encryption used to protect the data
D) Whether the data was encrypted at the time of the incident

**Correct Answer: B**

---

**Question 5:** Which VERIS timeline field captures how long it took the organization to detect the incident after it occurred?

A) `timeline.incident`
B) `timeline.exfiltration`
C) `timeline.discovery`
D) `timeline.compromise`

**Correct Answer: C**

---

**Question 6:** An organized crime group sends phishing emails to employees of a bank and uses the stolen credentials to log into the online banking system.
How many VERIS action types should be recorded?

A) One — only the phishing action matters
B) One — only the credential use matters
C) Two — social engineering AND hacking
D) Three — social engineering, hacking, and malware

**Correct Answer: C**

---

**Question 7:** What is the VCDB?

A) A commercial tool for security incident management
B) Verizon's internal database of breach cases used only for DBIR
C) A public, community-contributed dataset of VERIS-coded incidents
D) A government threat intelligence sharing platform

**Correct Answer: C**

---

**Question 8:** An employee deliberately copies 10,000 customer records to a personal USB drive before resigning.
Which VERIS actor type and action type best describe this incident?

A) Actor: External; Action: Hacking
B) Actor: Internal; Action: Misuse
C) Actor: Internal; Action: Error
D) Actor: Partner; Action: Misuse

**Correct Answer: B**

---

**Question 9:** Which VERIS asset type prefix is used to categorize a database server?

A) U
B) N
C) M
D) S

**Correct Answer: D**

---

**Question 10:** In VERIS, which action category covers the installation of keyloggers or backdoors on a compromised system?

A) Social
B) Hacking
C) Malware
D) Misuse

**Correct Answer: C**

---

**Question 11:** An organization suffers a ransomware attack that encrypts all files on 50 servers but no data is confirmed as exfiltrated.
Which VERIS CIA attributes should be recorded?

A) Confidentiality only
B) Integrity and Availability only
C) Confidentiality, Integrity, and Availability
D) Availability only

**Correct Answer: B**

---

**Question 12:** Which of the following is a key benefit of using a shared vocabulary like VERIS for incident recording?

A) It replaces the need for a SIEM in the SOC
B) It enables statistical comparison and trend analysis across organizations
C) It automatically generates incident response playbooks
D) It provides real-time threat intelligence feeds

**Correct Answer: B**

---

**Question 13:** Which publication uses VERIS-coded data as its primary data source for annual threat landscape statistics?

A) The NIST Cybersecurity Framework
B) The MITRE ATT&CK knowledge base
C) The Verizon Data Breach Investigations Report (DBIR)
D) The SANS Internet Stormcast

**Correct Answer: C**

---

**Question 14:** In VERIS, an attacker exploits an unpatched Apache vulnerability to gain root access to a web server.
What is the correct action type and variety?

A) Malware — Exploit code
B) Hacking — Exploit vulnerability
C) Error — Misconfiguration
D) Social — Phishing

**Correct Answer: B**

---

**Question 15:** A nation-state group compromises a defense contractor's network.
Which VERIS actor type would be most appropriate?

A) Internal — System admin
B) External — Nation-state
C) Partner — IT service provider
D) External — Organized crime

**Correct Answer: B**
