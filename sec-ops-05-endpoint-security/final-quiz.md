# Final Quiz: Endpoint Security Technologies

**Question 1**

A SOC analyst sees Windows Event ID 4624 with `LogonType: 3` and `AuthenticationPackageName: NTLM` for a Domain Administrator account, originating from an external IP address at 2 AM.
Which combination of factors makes this MOST suspicious?

A) LogonType 3 is always suspicious
B) NTLM authentication for a domain account at an unusual hour from an external IP
C) The event occurred at 2 AM
D) Domain Administrator accounts always generate Event 4624

---

**Question 2**

An analyst investigates a Sysmon Event 10 (ProcessAccess) alert.
The source process is `C:\Users\john\AppData\Local\Temp\svc.exe` accessing `C:\Windows\System32\lsass.exe` with `GrantedAccess: 0x1FFFFF`.
What is MOST likely happening?

A) A legitimate antivirus product is scanning for infections
B) Windows is performing a routine credential cache refresh
C) A credential dumping tool (e.g., Mimikatz) is attempting to extract password hashes
D) LSASS is terminating abnormally due to a crash

---

**Question 3**

Which of the following BEST describes a "Living Off the Land Binary" (LOLBin)?

A) Malware that disguises itself as a system file by copying itself to the Windows folder
B) A legitimate, signed system binary that attackers abuse to execute malicious actions
C) A binary that only exists in memory and never touches disk
D) An open-source offensive security tool bundled with Kali Linux

---

**Question 4**

In the Linux audit framework, the `auid` (audit UID) field has a value of `1000` in an event where `uid=0` and `euid=0`.
What does this tell a security analyst?

A) The event was generated by root performing a routine task
B) The user account with UID 1000 ran sudo or su to gain root access; the original login session was from UID 1000
C) The UID 0 and UID 1000 conflict indicates log corruption
D) User 1000 has been granted permanent root access

---

**Question 5**

An EDR platform generates an alert: `w3wp.exe (IIS Worker Process) spawned cmd.exe`.
The command line is `cmd.exe /c whoami`.
What type of attack does this MOST likely indicate?

A) An insider threat using the web server for administrative tasks
B) A web application command injection or web shell exploitation
C) A legitimate IIS health check running diagnostic commands
D) A scheduled maintenance script configured in IIS

---

**Question 6**

What is the key advantage of **XDR (Extended Detection and Response)** over traditional **EDR**?

A) XDR is less expensive than EDR
B) XDR uses machine learning while EDR does not
C) XDR correlates telemetry from multiple sources (endpoint, network, email, cloud) enabling cross-domain attack chain visibility
D) XDR does not require agent deployment on endpoints

---

**Question 7**

A threat hunter finds a Windows scheduled task named `\Microsoft\Windows\EdgeUpdate\ScheduledUpdate` that was created yesterday and runs PowerShell with a Base64-encoded command every 4 hours.
The legitimate `MicrosoftEdgeUpdate` task has a different path and XML signature.
Which MITRE ATT&CK technique is PRIMARILY being used?

A) T1059.001 — Command and Scripting Interpreter: PowerShell
B) T1053.005 — Scheduled Task/Job: Scheduled Task combined with T1036.004 — Masquerade Task or Service
C) T1105 — Ingress Tool Transfer
D) T1190 — Exploit Public-Facing Application
