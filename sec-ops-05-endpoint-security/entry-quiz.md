# Entry Quiz: Basic OS and Security Concepts

**Session 05 — Endpoint Security Technologies**
**Purpose:** Assess prerequisite knowledge before the session

**Time:** 10 minutes

**Format:** 5 multiple-choice questions

---

**Instructions:** Choose the single best answer for each question.

**Question 1**
