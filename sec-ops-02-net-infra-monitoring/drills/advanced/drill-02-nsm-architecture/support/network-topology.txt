GLOBALPORT LOGISTICS — NETWORK TOPOLOGY DIAGRAM
================================================
For use with Drill 02 (Advanced): Design a Complete NSM Architecture

This ASCII diagram shows the current network topology.
Use this as the baseline for your NSM architecture design.

Legend:
  [FW]    = Firewall
  [SW]    = Switch / Core switch
  [SRV]   = Server
  [WS]    = Workstation(s)
  [IoT]   = IoT / OT devices
  [TAP]   = Hardware TAP point (your design should place these)
  (SPAN)  = SPAN port opportunity
  ~~~     = WAN/Internet link
  ===     = High-speed LAN link (1 Gbps)
  ---     = Lower-speed LAN link (100 Mbps or SD-WAN)
  ...     = Wireless (Wi-Fi)
